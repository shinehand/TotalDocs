주요 기능

📄

HWP / HWPX 열람

한글 문서를 별도 프로그램 없이 크롬 브라우저에서 바로 열 수 있습니다.

✏️

직접 편집

텍스트 삽입·삭제·문단 분할 등 기본 편집 기능을 문서 위에서 직접 수행할 수 있습니다.

💾

저장 · 다운로드

편집한 내용을 원본 HWP/HWPX 형식으로 저장하거나 다운로드할 수 있습니다.

🔍

텍스트 검색

문서 내 키워드를 빠르게 검색하고 해당 위치로 이동할 수 있습니다.

🖼️

Canvas 기반 렌더링

WebAssembly 엔진으로 표·그림·수식·차트를 한컴 Viewer와 유사한 화질로 렌더링합니다.

📐

레이아웃 감사 패널

사이드바에서 현재 쪽의 표·그림·수식·차트 집계와 핫스팟 쪽 이동을 확인할 수 있습니다.

🌐 웹 뷰어 사용 방법

  1. 웹 뷰어 열기 위의 웹 뷰어 바로 열기 버튼을 클릭하거나 viewer.html 에 접속합니다. Chrome, Firefox, Safari, Edge 어디서든 작동합니다.
  2. HWP 파일 열기 화면 중앙에 .hwp / .hwpx / .owpml 파일을 끌어다 놓거나, 파일 선택 버튼으로 업로드합니다.
  3. 열람 및 편집 문서가 즉시 렌더링됩니다. 툴바의 편집 모드를 켜면 텍스트 편집도 가능합니다.
  4. 저장 편집 후 저장 또는 다른 이름으로 저장으로 HWPX·HTML·PDF 등으로 내보낼 수 있습니다.

🛠️ 크롬 확장 프로그램 설치 (선택사항 — 웹 페이지 HWP 링크 자동 열기 기능)

  1. 확장 프로그램 파일 다운로드 최신 릴리스 페이지 에서 TotalDocs-vX.X.X.zip 파일을 다운로드하고 압축을 해제합니다.
  2. 크롬 확장 프로그램 관리 페이지 열기 크롬 주소창에 chrome://extensions 를 입력하고 이동합니다.
  3. 개발자 모드 활성화 오른쪽 상단의 개발자 모드 토글을 켭니다.
  4. 압축 해제된 폴더 로드 압축해제된 확장 프로그램을 로드합니다 버튼을 클릭한 뒤, 압축을 해제한 폴더를 선택합니다.
  5. 설치 완료 크롬 툴바에 TotalDocs 아이콘이 나타나면 설치가 완료된 것입니다. 웹 페이지의 HWP 링크를 우클릭하면 뷰어로 바로 열 수 있습니다.

프로젝트 목표

TotalDocs는 어떤 HWP·HWPX·OWPML 문서라도 원본 형식 데이터에 따라 열리고, 한컴 Viewer와 같은 페이지 흐름·레이아웃으로 보이게 하는 것을 목표로 합니다.

현재 대표 샘플 5종에서 한컴 Viewer와 동일한 페이지 수를 달성했으며, 표·그림·수식·차트·폰트 렌더링 정밀도를 지속적으로 개선 중입니다.
